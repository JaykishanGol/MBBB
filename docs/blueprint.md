# **App Name**: MovieSpot

## Core Features:

- Movie Discovery: Browse and discover movies and TV shows using various filters from TMDB.
- Search Functionality: Find movies and TV shows by title, genre, or actor using the TMDB database.
- Bookmark Movies: Save the movies the user wants to watch in the future

## Style Guidelines:

- Dark gray background (#1A1A1A).
- Deep blue primary color (#2962FF) for trust and cinematic feel.
- Purple accent color (#BB86FC) for interactive elements.
- Plus Jakarta Sans font for body and headlines.
- Minimalist, geometric icons for navigation.
- Grid-based layout for movie posters and details.
- Subtle transition animations for loading and navigation.
